# Portal CMA

## Descrição

O Portal CMA é um hub centralizado que agrupa os principais links e sistemas utilizados no dia a dia, organizados em categorias para facilitar o acesso rápido e intuitivo.

Esta é a primeira versão (V0) do projeto, desenvolvida para ser uma solução leve e eficiente.

## Tecnologias Utilizadas

*   **HTML5:** Para a estrutura semântica do portal.
*   **Tailwind CSS:** Para a estilização ágil e responsiva da interface.
*   **JavaScript:** Para as funcionalidades interativas, como a busca e a área administrativa.

## Funcionalidades

*   **Busca Inteligente:** Filtra os links em tempo real, desconsiderando acentos e diferenças entre maiúsculas e minúsculas.
*   **Área Administrativa Oculta:** Uma coluna especial ("Administrativo") fica oculta por padrão para usuários comuns.
*   **Layout Responsivo:** A interface se adapta a diferentes tamanhos de tela, de desktops a dispositivos móveis.

## Acesso à Área Administrativa

A coluna "Administrativo" contém links de acesso restrito e, por padrão, não é visível. Para exibi-la, siga os passos:

1.  Clique no ícone de perfil no canto superior direito da tela.
2.  Quando solicitado, informe o código de acesso: **CMA2025**
3.  A coluna administrativa será exibida. Para ocultá-la novamente, basta clicar no ícone de perfil e confirmar a saída.
